# Apex Finance (Core Backend Architecture)

A local-first, high-performance digital finance diary featuring Zero-Based Budgeting and Predictive Analytics.

## System Architecture
* **Execution Environment:** Python 3 Virual Machine
* **Data Persistence:** SQLite C-API
* **Validation Layer:** Pydantic (Rust-Engine type casting)
* **Frontend UI:** CustomTkinter (Event-driven OS rendering)
